## 中文

v0.3.3 - 热键热重载修复

### 关键修复

**热键配置修改后立即生效**
- 修复点击 Apply 后所有热键失效的问题
- 修复删除的热键仍然触发录音的"幽灵热键"问题
- 在设置中修改热键配置后，无需重启即可生效

**系统稳定性改进**
- 修复 HotkeyService 生命周期管理错误
- 优化线程同步，避免竞态条件
- 移除冗余的热键注册代码

### 主要功能

**本地转录引擎**
- sherpa-onnx 轻量级引擎（安装包 <250MB）
- 双模式流式转录（chunked/realtime）
- 纯 CPU 推理，无需 GPU

**云端转录服务**
- Groq: whisper-large-v3-turbo，快速准确
- SiliconFlow: FunAudioLLM/SenseVoiceSmall
- Qwen ASR: qwen3-asr-flash，极速转录

**智能输入系统**
- 多种输入方式支持
- 增量式文本替换
- 剪贴板智能恢复

## English

v0.3.3 - Hotkey Hot-Reload Fix

### Key Fixes

**Hotkey Configuration Changes Take Effect Immediately**
- Fixed issue where all hotkeys stopped working after clicking Apply
- Fixed "ghost hotkey" issue where deleted hotkeys still triggered recording
- Hotkey configuration changes now apply instantly without restart

**System Stability Improvements**
- Fixed HotkeyService lifecycle management errors
- Optimized thread synchronization to avoid race conditions
- Removed redundant hotkey registration code

### Key Features

**Local Transcription Engine**
- sherpa-onnx lightweight engine (installer <250MB)
- Dual-mode streaming transcription (chunked/realtime)
- Pure CPU inference, no GPU required

**Cloud Transcription Services**
- Groq: whisper-large-v3-turbo, fast and accurate
- SiliconFlow: FunAudioLLM/SenseVoiceSmall
- Qwen ASR: qwen3-asr-flash, ultra-fast transcription

**Smart Input System**
- Multiple input method support
- Incremental text replacement
- Smart clipboard recovery
