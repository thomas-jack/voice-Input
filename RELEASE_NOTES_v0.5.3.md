# v0.5.3 - 应用级多语言与字体优化

## 中文

v0.5.3 - 应用级多语言与字体优化

### 主要更新

**UI 多语言与即时切换**
- 设置页新增 UI 语言选择，支持系统语言自动跟随
- 中英文界面覆盖设置、主窗口、托盘菜单与对话框
- 语言切换立即生效（部分菜单需重新打开刷新）

**中文显示优化**
- 内置开源圆体字体 Resource Han Rounded CN（OFL 许可）
- 应用级默认字体与 Hinting 优化，提升不同缩放下的清晰度

### 升级方式

**从 v0.5.2 升级**
1. 下载 SonicInput-v0.5.3-win64.exe
2. 关闭旧版本，替换可执行文件
3. 启动应用（配置自动沿用）

**注意事项**
- 与 v0.5.2 配置完全兼容
- 无需手动迁移设置
- 直接替换即可

### 系统要求

**最低**
- Windows 10 64-bit
- Intel Core i3 或同级 CPU
- 4GB RAM
- 500MB 磁盘空间

**推荐**
- Windows 11 64-bit
- Intel Core i5 或 AMD Ryzen 5
- 8GB RAM
- 1GB 磁盘空间

### 技术支持

**配置与日志**
- 配置:`%AppData%\SonicInput\config.json`
- 日志:`%AppData%\SonicInput\logs\app.log`
- 模型:`%AppData%\SonicInput\models/`

**问题反馈**
- GitHub Issues: [提交 Issue](https://github.com/Oxidane-bot/SonicInput/issues)

## English

v0.5.3 - App-wide Localization and Font Polish

### Key Improvements

**App-wide localization**
- UI language selector in Settings with system-locale default
- Coverage across Settings, main window, tray menu, and dialogs
- Live language switching for open windows (menus refresh on reopen)

**Chinese UI font polish**
- Bundled Resource Han Rounded CN (OFL) for a more modern rounded look
- App-level default font and hinting for sharper CJK rendering at common DPI scales

### Upgrade Steps

**From v0.5.2**
1. Download SonicInput-v0.5.3-win64.exe
2. Close old version, replace executable
3. Launch app (config auto-applied)

**Notes**
- Fully compatible with v0.5.2 config
- No manual settings adjustment needed
- Direct replacement works

### System Requirements

**Minimum**
- Windows 10 64-bit
- Intel Core i3 or equivalent CPU
- 4GB RAM
- 500MB disk space

**Recommended**
- Windows 11 64-bit
- Intel Core i5 or AMD Ryzen 5
- 8GB RAM
- 1GB disk space

### Technical Support

**Configuration and Logs**
- Config: `%AppData%\SonicInput\config.json`
- Logs: `%AppData%\SonicInput\logs\app.log`
- Models: `%AppData%\SonicInput\models/`

**Issue Reporting**
- GitHub Issues: [Submit Issue](https://github.com/Oxidane-bot/SonicInput/issues)
